# GraphQL-todo
練習 Apollo Server v4 + MongoDB 後端建構  
📚 主要目的是學習 GraphQL，並記錄自己實際有動手做過！

---

## 🙋‍♂️ 說明

此專案為個人練習用途，無部署、無使用者介面。  
功能包含基本的 CRUD 操作：新增、查詢、更新、刪除 Todo。

---

## ✅ 技術棧

- Node.js + Express
- Apollo Server v4 (GraphQL)
- MongoDB + Mongoose
